# SalaryCalculator.Umbrella
##  Developer Environment

###  Requirements

- Erlang 23.0
- Elixir 1.11.0

### Install Dependencies

$ mix deps.get
 

### Run Server

$ mix phx.server



##  Production

###  Requirements

- Fedora 33


### Run Server

$ tar xzvf salary_calculator_web-1.0.0.tar.gz
$ ./bin/salary_calculator_web start
